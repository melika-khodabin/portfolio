**Tech stack**

This project follows a standardized and opinionated front-end stack to ensure consistency, maintainability, and modern best practices across the codebase.

- Yarn@v1.22 – Preferred package manager for dependency management and scripts.
- Vite – Fast and modern build tool for local development and bundling.
- TypeScript – Statically typed JavaScript for improved code reliability and scalability.
- Prettier – Code formatter to enforce consistent styling.
- Tailwind CSS – Utility-first CSS framework for rapid UI development.
- Daisy-UI – Tailwind CSS plugin for pre-styled UI components.
- Next.js – Core UI library for building interactive UIs.
- Jotai@^2 – Primitive and flexible state management solution.
- Lodash – Utility library for common operations.
- lucide-react – Collection of popular icon packs as React components.
